# Time Complexity Practice (Single Loops)

## Easy Problems

### Question 1
```js
for (let i = 1; i < n; i *= 2) {
  // O(1) work
}
```
### Answer:
O(log n)


### Question 2 
```js
for (let i = 0; i < 10; i++) {
  // O(1) work
}
```

### Answer:
O(1)
